JR Recovery & Process Service - Frontend Prototype 🕵️‍♂️🚔

A high-tech, highly interactive, and fully responsive frontend prototype for JR Recovery Process Service, based in Upstate New York. Designed to convey authority, discretion, and a tactical edge, this landing page is specifically tailored for a business led by a former police officer specializing in process serving, skip tracing, and asset recovery.

✨ Key Features & Interactivity

This is not a standard static template. The prototype includes several dynamic, gamified elements designed to make the user experience highly memorable:

Gamified "Skip Trace" Entrance: Users are greeted by a dark, tactical radar sweep overlay. They must click the flashing "target blip" to bypass the security wall and access the main website.

Tactical Coverage Map Modal: Clicking on the service counties (St. Lawrence, Jefferson, Franklin) opens a dark UI modal. It features a stylized, glowing SVG map outline of the selected county, an active radar sweep, and dynamically generated red "target blips" to simulate a live tracking system.

Modern, Authoritative UI: A sleek, dark-slate hero section transitioning into clean, highly readable service areas. Features custom scrollbars, subtle hover animations, and responsive layouts for mobile and desktop.

🤫 Hidden Easter Eggs

To add a signature touch, the site includes hidden features:

The "Undercover" Police Lights: * How to trigger: Rapidly click the shield logo/site title in the top-left navigation bar three times.

Effect: A full-width red and blue police strobe light flashes across the top of the browser window.

The C4:Technologies Deep Portal:

How to trigger: Scroll to the bottom footer and hover over the "Developed by C4:Technologies" link to reveal an X-Ray scanner effect. Click the button.

Effect: The screen locks into a massive cyan radar system, complete with a live-streaming terminal of "Recovered Digital Assets." Includes a direct exit link to the C4 Development Center and an abort switch to return to the main site.

🛠️ Technology Stack

Structure: Vanilla HTML5

Styling: Tailwind CSS (via CDN for rapid prototyping without build steps)

Icons: FontAwesome 6 (Free)

Logic & Animations: Vanilla JavaScript and CSS Keyframes (@keyframes) for radar sweeps, glowing blips, and modal transitions.

🚀 Quick Start & Deployment

Because this prototype is entirely self-contained in a single file, deployment and testing are incredibly simple.

Local Testing

Clone this repository or download the source code.

Double-click the index.html file to open it directly in any modern web browser (Chrome, Safari, Firefox, Edge). No local server required!

Vercel Deployment

This repository is pre-optimized for instant deployment on Vercel:

Push this code to a GitHub/GitLab repository.

Log into Vercel and click Add New > Project.

Import your repository.

Leave the Framework Preset as Other and click Deploy.

Your site will be live on a .vercel.app domain in seconds.
